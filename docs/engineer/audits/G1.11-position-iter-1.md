## Material differences persisting (post-iter-1)

1. **Overtake detection (alto impacto)**: CC tiene un sistema completo de detección de adelantamientos con gap analysis (muestras cada 1s, promedio de ventana, min gap para pass, cooldown 20s). Vantare no tiene nada. Sin esto, el spotter gana/overtake está ciego. **Implementar en iter-2**.
   - Gate: solo en `SessionPhase.Green`, `SessionType.Race`, `CompletedLaps > 0`
   - Clean pass check: damage/offtrack (10s) y yellow (3s) invalidan
   - Mismo-class check antes de reportar
   - `minAverageGapForPassMessage`, `passCheckSamplesToCheck`

2. **Race start messages (alto impacto)**: Vantare no evalúa pérdida/ganancia de posiciones en la salida. CC dispara terrible/bad/good/ok start basado en `SessionStartClassPosition` vs `ClassPosition` actual. **Implementar en iter-2**.

3. **Position validation (medio impacto)**: CC invalida mensajes si la posición cambió mientras estaba en cola, o si entramos en pits. Vantare no tiene validación. **Implementar en iter-2**.

4. **Last place detection (bajo impacto)**: CC detecta cuando el jugador es último (`isLastInStandings`) y cuenta laps en last place para "consistently last". **Iter-3**.

## Non-material differences (accepted)

- Vantare usa ms para cooldown (10_000ms), CC usa TimeSpan (20s). El valor exacto puede converger en iter-2.
- Vantare no necesita distinction leading/pole/quickest hasta que haya voice catalog.
- Vantare no tiene `folderDriverPositionIntro` para prefijo de posición (solo CC non-English).
- `frequencyOfOvertakingMessages` y `frequencyOfBeingOvertakenMessages` son user settings; Vantare los hardcodea a valores por defecto en iter-1.

## Bloqueos

- `ClassPosition` no está disponible en el modelo de telemetría de Vantare (solo `Place` = overall). LMU Shared Memory expone `mPlayerTicksPosition` (overall) y `mPlayerClassPosition`. **GAP**: leer `mPlayerClassPosition` del buffer LMU (offset `0x0A4` en LMU parser si está mapeado). Sin class position, no podemos tener parity en position messages intra-clase.
- `SessionStartClassPosition` no está mapeado en Vantare telemetry model.
