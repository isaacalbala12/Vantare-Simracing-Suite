## Notas

- Esta fase es principalmente "toggle + formato + columna": muchas features pequeñas y repetitivas.
- Lo más complejo es el motor de reglas de visibilidad condicional.
- Se implementa sobre datos normalizados, por lo que funciona para todos los sims soportados.
