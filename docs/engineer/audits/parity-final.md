## What's Blocked vs Code-Pure

### BLOCKED by LMU data / shared memory offsets

These features require a telemetry field that does not currently exist in LMU's shared memory or has not been verified:

| Feature | Blocking field | CC source |
|---|---|---|
| FCY sub-phases (pits closed, open, last lap, etc.) | `YellowFlagState` / `FCYPhase` enum — not in LMU session buffer | `FlagsMonitor.cs:683-767` |
| Oil pressure warning | `mOilPressureWarning` — placeholder offset, no verified data | `EngineMonitor.cs:107-112` |
| Fuel pressure warning | `mFuelPressureWarning` — no verified offset | `EngineMonitor.cs:113-118` |
| Brake temps (cold/hot/cooking) | No brake temperature data in LMU telemetry | `TyreMonitor.cs:52-63` |
| Tyre pressure (very high/high/OK/low/very low) | No per-wheel pressure in LMU telemetry | `TyreMonitor.cs:206-236` |
| Tyre inner/middle/outer temps (camber analysis) | No IMO temp data | `TyreMonitor.cs:181-203` |
| Per-corner locking/spinning warnings | Requires track landmarks + wheel slip data per tick | `TyreMonitor.cs:151-162` |
| Stock car rules (lucky dog, wave around, etc.) | No `StockCarRulesData` equivalent in LMU | `FlagsMonitor.cs:416-543` |
| R3E pit menu actions | R3E-specific shared memory, not applicable to LMU | `PitStops.cs:96-114` |
| Driver stint seconds remaining | No `DriverStintSecondsRemaining` in LMU | `DriverSwaps.cs:56-109` |
| Opponent best lap / sector times | Requires per-opponent lap data not yet available | `Opponents.cs:259-327` |
| Pit box location / distance countdown | Requires world position or pit-box offset working correctly | `PitStops.cs:329-545` |
| Mandatory pit window (open lap/time) | No `HasMandatoryPitStop` / `PitWindowStart` / `PitWindowEnd` in LMU | `PitStops.cs:617-817` |
| Track landmarks / corner names | LMU does not expose corner name mapping | `Timings.cs:282-287` |
