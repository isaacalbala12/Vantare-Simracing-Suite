# AUTH-04 — Registro de usuarios y recuperación de password

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Añadir registro de nuevos usuarios (`signUp`) y recuperación de password (`resetPassword`) desde la app para que la beta pública permita adquirir usuarios sin intervención manual.

**Architecture:** Exponer `signUp` y `resetPasswordForEmail` en `supabase-auth.ts` (wrappers del SDK de Supabase Auth ya conectado), añadir un toggle "Crear cuenta" / "Iniciar sesión" en `LoginScreen`, y un enlace "¿Olvidaste tu contraseña?". El login existente no se toca; el cambio es puramente aditivo.
> **⚠️ DECISIÓN CERRADA (A, 2026-07-06):** signup **abierto + confirmación por email**. Tras `signUp` el usuario recibe `session: null` y la app muestra "Revisa tu email" (no inicia sesión automática). El toggle "Crear cuenta" / "Iniciar sesión" es público. No es invite-only. Esta decisión estaba pendiente en el handoff §5.#4 y se confirma aquí.

**Tech Stack:** React 19, TypeScript, `@supabase/supabase-js` (ya instalado), Vitest + Testing Library, Wails runtime.

```typescript
export async function signUp(
  email: string,
  password: string,
): Promise<{ user: User | null; session: Session | null; error?: string }> {
  try {
    const c = getSupabaseClient();
    const { data, error } = await c.auth.signUp({ email, password });
    if (error) return { user: null, session: null, error: error.message };
    return { user: data.user, session: data.session, error: undefined };
  } catch (err) {
    if (isConfigError(err)) return { user: null, session: null, error: missingConfigError };
    return { user: null, session: null, error: err instanceof Error ? err.message : "Error desconocido" };
  }
}
```

```typescript
export async function resetPasswordForEmail(
  email: string,
): Promise<{ error?: string }> {
  try {
    const c = getSupabaseClient();
    const redirectTo = `${oauthCallbackUrl()}/#/auth/callback`;
    const { error } = await c.auth.resetPasswordForEmail(email, { redirectTo });
    if (error) return { error: error.message };
    return { error: undefined };
  } catch (err) {
    if (isConfigError(err)) return { error: missingConfigError };
    return { error: err instanceof Error ? err.message : "Error desconocido" };
  }
}
```

```typescript
const handleSignUp = useCallback(async (e: FormEvent) => {
  e.preventDefault();
  setError(null);
  setSubmitting(true);
  try {
    const { session, error: signUpError } = await signUp(email, password);
    if (signUpError) { setError(signUpError); return; }

    // Email confirmation pending
    setSignupEmailSent(true);
  } catch {
    setError("Error al crear la cuenta. Intenta de nuevo.");
  } finally {
    setSubmitting(false);
  }
}, [email, password, onLoggedIn]);
```
