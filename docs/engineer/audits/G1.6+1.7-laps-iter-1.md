**Gates — LapCounter:**
- Solo en `SessionType.Race`, `IsNewLap`, `CompletedLaps > 0`, `SessionPhase != Finished`
- `SessionLapsRemaining == 2`: disable pearls + position check
- `SessionLapsRemaining == 1` o `IsLastLap`: position check:
  - `position == 1` → `last_lap_leading`
  - `position < 4` → `last_lap_top_three`
  - `position > 4` → `last_lap_EU` o `white_flag_last_lap`
- `SessionLapsRemaining == 2`:
  - `position == 1` → `two_to_go_leading`
  - `position < 4` → `two_to_go_top_three`
  - `position >= start+5 && slower lap` → `two_to_go` con BAD pearl
  - `position >= 4` → `two_to_go` con NEUTRAL pearl

**Cooldowns:**
- `lastFinishMessageTime = DateTime.MinValue` (para finished)
- `lastSessionEndMessagesPlayedAt.AddSeconds(10)` dedup en SessionEndMessages
