# LMU weekly calendar hotfix changelog template

Usar esta plantilla en `docs/changelog.md` cuando se publique un hotfix semanal de calendario.

```markdown
## [vX.Y.Z.N] - YYYY-MM-DD

### Fixed
- Actualizado calendario oficial LMU semanal desde `YYYY-MM-DD`.

### Calendar
- Beginner / Bronze SR: `Serie 1`, `Serie 2`, `Serie 3`.
- Intermediate / Silver SR: `Serie 1`, `Serie 2`, `Serie 3`.
- Advanced / Gold SR: `Serie 1`, `Serie 2`, `Serie 3`.
- Weekly: `Serie weekly` con slots UTC oficiales.

### Notes
- Las sesiones de practica y qualy son estimadas por Vantare: practica 3 min + qualy 8 min.
- La duracion de carrera se toma del calendario oficial LMU.
- No se han anadido features nuevas en este hotfix.
```

## Reglas

- No prometer features nuevas.
- No mencionar datos no incluidos en el texto oficial.
- No anadir precios, ratings, votos ni URLs de inscripcion inventadas.
- Si no hay version publicada todavia, dejar la entrada como `Unreleased` hasta el release.
